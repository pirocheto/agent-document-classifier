# Agent for Document Classification

This project implements an AI agent designed to classify documents into predefined categories based on their content. The agent leverages advanced language models and is built using FastAPI for the web framework.

## Stack

- FastAPI: A modern, fast (high-performance) web framework for building APIs with Python 3.13+ based on standard Python type hints.
- LangChain: A framework for developing applications powered by language models.
- AgentCore: AWS Service for deploying and managing AI agents.
